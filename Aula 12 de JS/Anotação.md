*Resumo da aula 12 de JS*

* Condições aninhadas
é uma condição composta com mais consições.
ex de condição aninhada:

if (cond1) {
    (BLOCO1)
} else {
    if (cond2) {
        (BLOCO2)
    } else {
        (BLOCO3)
    }
}

*Condição MUltipla*

- muito importante para valores fixos, ela nao tem só a possibilidade do sim e não.
ela tem a possibilidade de outros valores fixos.

switch(expressão){
    case valor 1:

    break
    case valor 2:

    break
    case valor 3:

    break
    default:

    break

}

*IMPORTANTE*
- 'BREAK' no final de cada (case) é obrigatório se não vai dar erro e vai jogar todas as respostas do comando.
